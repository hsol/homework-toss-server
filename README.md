# The Team Showcase

![The Team Showcase](documents/the_team_showcase.png)

## prepare
### dependency
#### python modules
- pip
- sphinx
- django
- django-filters
- corsheaders
- MySQLdb
### commands

    2. python manage.py migrate

## execute
